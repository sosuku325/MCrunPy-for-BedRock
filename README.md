# MCPyRunner

**MCPyRunner** は、Pythonで実装されたMinecraft (PaperMC) サーバーの自動プロビジョニングおよび起動ツールです。
Java実行環境のセットアップから、Paperサーバーモジュールのダウンロード、そしてサーバープロセスの起動に至るまでの一連のワークフローを完全に自動化します。
---

## 🚀 特徴

* ☕ Java（OpenJDK 21）を自動ダウンロード＆セットアップ
* 📦 PaperMCの最新ビルドを自動取得
* 📜 EULA自動同意
* 🎮 サーバーを即起動可能
* 💻 コンソール操作対応（コマンド入力OK）

### 🎮 Minecraftバージョンを変更する方法
```python
PAPER_VERSION = "1.21.11"
```
